Your Food, Your Mood

Клиентская часть веб-сервиса для хранения, поиска и фильтрации кулинарных рецептов.

Функциональность

- поиск рецептов через TheMealDB;
- просмотр рецепта;
- фильтрация по категориям и времени;
- создание пользовательских рецептов;
- добавление фото рецепта и фото приготовления;
- избранное;
- история просмотров;
- общие лайки через Supabase;
- realtime-обновления;
- русская и английская локализация;
- темная тема.

Технологии

- React
- Vite
- JavaScript ES6+
- CSS / БЭМ
- Supabase
- TheMealDB API
- MyMemory Translation API
- GitHub Pages

Запуск проекта

npm install
npm run dev

Сборка

npm run build
npm run preview
